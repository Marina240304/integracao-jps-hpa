# Integração entre JPS e HPA*

Projeto desenvolvido para a disciplina de Metodologias e Técnicas da Computação do Mestrado em Ciência da Computação da UFSM.

## Autoras

- Larissa Alves da Paixão
- Marina Bolzan Medeiros Teixeira

## Algoritmos implementados

- A*
- Jump Point Search (JPS)
- HPA*
- HPA-JPS

Na integração proposta, o JPS é utilizado nas buscas locais da estrutura hierárquica do HPA*.

## Requisitos

- Python 3.11 ou superior

pip install -r requirements.txt
